==============================================
ShortQut --- the GPS that Learns About Traffic
==============================================

ShortQut is a GPS system that offers turn-by-turn navigation while
automatically routing the user around areas with probable congestion.
The more you use ShortQut, the better it becomes at predicting which
areas to avoid. ShortQut users are encouraged to share their collected
data with other ShortQut users to improve the accuracy of the results
for everyone.
